## 3. 게이트 판정 및 수정 지시

### 게이트: FIX (전체)

Blocker가 1개 이상 잔존하는 초안은 클라이언트에게 보여줄 수 없다. 아래 수정 우선순위대로 처리 후 재검토.

---

### 코코댕-순순스틱-v2 — FIX 수정 지시 (P0 3건)

**P0-1. BRAND LOGO placeholder 제거 (s00)**
- "BRAND LOGO" 텍스트 즉시 삭제
- 실제 로고 파일이 없으면 빈 공간으로 두거나 제품명 서체로 대체 — placeholder 문자열 노출은 절대 불가

**P0-2. broken-img 아이콘 3개 교체 (s01 다크 섹션)**
- 코코넛 과육·식이섬유·기호성 아이콘: SVG 인라인 삽입 또는 텍스트 기반 뱃지로 교체
- 이미지 URL 미확정 시 CSS 그리드+텍스트 카드로 fallback 처리 — broken-img 노출 불가

**P0-3. 헤드라인 오타 수정 (s04)**
- 'WHY 순순스틱을 선택하는 s가지 이유' → '5가지' 또는 '여섯 가지' 등 실제 개수 한글/숫자로 수정

---

### 대관령황태이야기 — FIX 수정 지시 (P0 2건)

**P0-1. s01 패키지 히어로 섹션 공백 해소**
- 현재 제품 스틱 이미지 상하로 대형 암흑 공간만 남아 있음
- 옵션 A: 대형 DAEGWALLYEONG 레터링을 배경 레이어로 삽입해 공간 활용
- 옵션 B: 성분 키워드(황태/청국콩) 타이포 그래픽 병치
- 옵션 C: 섹션 상하 padding을 현재의 40% 이하로 압축
- 어떤 방식이든 '미완성 칸'으로 읽히지 않는 구조로 재설계 필수

**P0-2. s05 클로징 팔레트 수정**
- 웜 타우프/그레이 배경 → 페이지 메인 팔레트(다크 네이비 계열 또는 코발트 라이트 틴트)로 교체
- 마지막 화면에서 브랜드 컬러 시스템이 무너지는 것은 첫인상만큼 치명적

---

### 주식회사럽앤다이브 — FIX 수정 지시 (P0 2건)

**P0-1. s02 하단 구조적 공백 제거**
- 이미지가 display:none 처리된 컨테이너의 높이/패딩 제거
- 섹션을 텍스트 전용 레이아웃으로 재구성하거나, 이미지 자리에 텍스처/패턴 배경으로 fill

**P0-2. s03 TEXTURE·SCENT 섹션 재설계**
- 이미지 제거 후 각 블록에 대형 크림 공백이 그대로 노출됨
- 텍스트 전용 풀블리드 레이아웃으로 재구성: 헤드라인 타이포 스케일 업 + 배경 컬러 블록으로 섹션 분리
- 또는 CSS 텍스처(grain, linear-gradient)로 이미지 영역 대체

---

### 모모스커피-sand-luxury-r2 — FIX 수정 지시 (P0 1건)

**P0-1. POINT 1·2·3 다크 섹션 구조 재설계 (s03 하단~s04 상단)**
- 이미지 없는 상태에서 min-height/padding이 그대로 남아 헤드라인 텍스트만 달랑 노출됨
- 처리 방법 (택1):
  - A: 섹션 padding을 text-only 모드로 압축 (상하 각 48px 이내), 배경 텍스처(원두 패턴 SVG 등)로 공간 채움
  - B: 3개 POINT를 수평 카드 3열로 재편 — 이미지 없이도 완결된 레이아웃
  - C: POINT별 대형 골드 숫자 타이포를 배경 레이어로 삽입해 공간 활용
- 현 상태로는 레이아웃 파손으로 읽힘
