표본 48프레임(장식 상위 + 계열 다양성 + 장식 에셋 프레임 6장)을 9개 분석 패널로 분류,

용도: **씬 컴포지션 스프린트(7씬×2000px)의 사양서.** 트랙 D 유틸 22종([asset-density-sprint.md](asset-density-sprint.md))의
보강 규칙 + 신규 이식 목록 + 정량 밀도 기준을 공급한다. 원본 에셋은 복제하지 않는다 — 문법(무엇을·어디에·어떤 비례로)만

## 1. 밀도 정량 룰 (184종 실템플릿 186프레임 실측)

| 지표 | 중앙값 | 상위 25% | 시스템 목표(초판) |
|---|---|---|---|
| 프레임당 장식 노드 | 4.0 | 14.0 | 씬당 **6~14** (상한: 가독 충돌 시 감사가 지적) |
| 장식/1000px(세로) | 2.5 | 8.7 | **3~9** |
| 폰트 크기 대비(fs_max/fs_min) | 3.0 | 3.6 | **≥3.0** (타이틀 40~64 ÷ 본문 14~17 = 2.9~4.6과 정합) |
| 프레임당 색 수 | 5 | — | ≤7 (토큰 팔레트 내) |

- 장식 최다 프레임은 어워드·구성 페이지 계열(노드 500~675) — 장식의 60~80%가 **배경 레이어**에 몰림.
- 밀도는 균등 살포가 아니라 **중앙 집중형 원형 존**(프레임폭 40~55%)에 한정 — 텍스트 가독과 충돌하지 않는 제어 원칙.

## 2. 3단 z축 레이어링 원칙 (전 프레임 공통 구조)

```
d:1-2  배경 발광층 — 방사형 코로나·글로우·스타필드 (프레임폭 40~160%, 저채도·저불투명)
d:3-5  중간 장식층 — 월계수·파티클·컨페티·도형 산란 (콘텐츠 주변 침범 10~26%)
d:6-8  전면 정보층 — 배지·칩·말풍선·KPI·텍스트 (완전 불투명, 토큰 원색)
```
씬 래퍼(`.scene`)가 이 3층의 캔버스. 장식은 콘텐츠 아래 z-index 고정, 씬 밖 누출 금지(overflow:hidden).

## 3. 토큰 바인딩 표준

- 색 3단: `--accent`(강조 fill·gradient 중심) → `--brand`(구조 배경·씰 내원) → `--ink`(아이콘 단색·실루엣).
- 형태 언어 5종 바인딩: soft-round(카드·배지·말풍선), sharp-editorial(배너·KPI·블록 화살표),
  organic(파티클·월계수·손그림), arch-serif(인용·씰 TEXT_PATH), neutral(아이콘 시트 기본).
- 가드레일은 트랙 D와 동일하게 토큰 상한으로 강제(그레인≤.08, 해칭≤.04, 골드 스포트라이트≤.22, conic 링 페이지 1회 등).

## 5. 자체 SVG 에셋 시트 6계열 (제너레이터 제작 가치 확인)

184종 CANVAS의 장식 에셋 프레임 6장(018~023)에서 계열별 변형 파라미터를 역설계:

1. **화살표** — 4계열 39변형(손그림 18·직선 12·블록 8·루프 1) × 4방향 파생
2. **체크마크** — 3계열(V획 stroke 6종·도트 궤적·배지 8변형)
3. **말풍선** — 꼬리 4방향 × 단색/outline × 타이핑/중첩 = 49변형
4. **시계·타이머** — 아날로그·스톱워치·모래시계, 인라인(40~80px)·배너(200~320px) 2스케일
5. **클릭 커서** — 포인터 미세변형 28종 + 클릭 물결 동심원
6. **위치·교통** — 핀 4색×2종 + 건물 실루엣 모듈 + 교통

전부 파라미터릭 생성 가능(수작업 path 불필요 수준의 단순 지오메트리) → `--ink/--accent` currentColor 바인딩으로
프로젝트 팔레트 자동 분화. 화살표·말풍선·체크·클릭이 표본 frequency의 60% 이상 — 이 4계열이 최우선.
