# 디자이너 편집용 결과물 — 방안 비교

> 현재 PDF는 레이어가 합쳐져 편집이 어렵습니다.
> 디자이너가 실제로 수정 가능한 형태의 결과물 3가지 방안을 비교합니다.

---

## 방안 A. PSD 파일 (포토샵)

### 결과물
포토샵에서 바로 열 수 있는 `.psd` 파일

### 구조 (포토샵 레이어 패널)
```
📁 01_히어로
   🖼 배경 이미지
   T  "인생 소금빵"          ← 텍스트 더블클릭 → 즉시 수정
   T  "천연발효종 24시간..."

📁 02_브랜드 스토리
   🖼 스타일링 이미지
   T  제목
   T  본문

📁 03_핵심 특징
   🎨 배경
   T  항목 제목 / 본문

... (섹션별 8~9개 그룹)
```

### 디자이너 작업 방식
- PSD 더블클릭 → 포토샵에서 바로 열림
- **텍스트 레이어 클릭 → 즉시 수정** (폰트, 크기, 색상 변경 가능)
- 이미지 레이어 선택 → 교체, 이동, 크기 조절
- 섹션(그룹) 순서 변경, 삭제, 추가

### 제약 사항
- 포토샵 필수
- CSS 효과(그림자, 둥근 모서리 등)는 이미지로 대체
- 사용된 한글 폰트가 디자이너 PC에 설치되어 있어야 함 (폰트 파일 함께 제공)

### 예상 파일 크기
30~50MB / 건

---

## 방안 B. 레이어 분리 키트 (PNG + 가이드)

### 결과물
섹션별 이미지 + 원본 소재 + 배치 가이드 JSON이 담긴 ZIP 파일

### 구조 (폴더)
```
📁 sections/       ── 섹션별 통합 이미지 (8~9장)
📁 backgrounds/     ── 배경만 분리한 이미지
📁 photos/          ── 스타일링 촬영 이미지 원본 (6장)
📁 layers/          ── 히어로, 브레이크 등 레이어 이미지
📁 fonts/           ── 사용 폰트 전체
📋 manifest.json    ── 각 요소의 위치, 크기, 폰트, 색상 정보
📋 style-guide.json ── 색상 팔레트, 폰트 조합
```

### 디자이너 작업 방식
1. manifest.json에서 전체 구조와 스타일 확인
2. 포토샵/피그마/일러스트레이터 등 **아무 도구**에서 새 파일 생성
3. 배경 이미지 배치 → 텍스트 직접 입력 → 소재 이미지 배치
4. **조합 자유도 최고** — 원하는 대로 재구성 가능

### 제약 사항
- **수동 조합 필요** — 디자이너가 직접 배치해야 함
- 텍스트는 이미지가 아닌 가이드 정보만 제공 → 직접 타이핑
- PSD보다 작업 시간이 더 소요

### 예상 파일 크기
60~80MB / 건

## 방안 C. Figma Import용 SVG

### 결과물
Figma에서 바로 import 가능한 섹션별 `.svg` 파일 + 이미지 소재

### 구조 (폴더)
```
📁 sections/     ── 섹션별 SVG 파일 (8~9개)
📁 photos/       ── 이미지 소재
📁 fonts/        ── 사용 폰트
📋 import-guide  ── Figma import 방법 안내
```

### 디자이너 작업 방식
1. Figma → File → Import → SVG 파일 선택
2. **텍스트가 편집 가능 상태로 유지** → 더블클릭해서 수정
3. 이미지 교체 가능
4. Figma 컴포넌트/Auto Layout으로 전환 가능

### 제약 사항
- **Figma 또는 Illustrator 필수** (포토샵에서는 벡터가 래스터로 변환)
- 복잡한 레이아웃(그리드, 카드 등)의 재현 정확도가 PSD보다 낮을 수 있음
- 디자이너 PC에 한글 폰트 설치 필요

### 예상 파일 크기
20~40MB / 건

---

## 비교 요약

|  | A. PSD | B. 레이어 키트 | C. Figma SVG |
|--|--------|---------------|-------------|
| **바로 편집 가능?** | ✅ 즉시 | ❌ 조합 필요 | ✅ import 후 즉시 |
| **텍스트 수정** | ✅ 레이어 클릭 | ❌ 직접 타이핑 | ✅ 더블클릭 |
| **이미지 교체** | ✅ | ✅ | ✅ |
| **필요 도구** | Photoshop | 무관 | Figma |
| **작업 속도** | 빠름 | 느림 | 빠름 |
| **자유도** | 높음 | 최고 | 높음 |
| **구현 소요** | 3~4일 | 2일 | 2~3일 |

---

## 참고

- 어떤 방안을 선택하든 **현재 웹 미리보기 (HTML)는 그대로 유지**됩니다.
- 기업(클라이언트)에게는 기존처럼 PNG 미리보기 + 다운로드가 제공됩니다.
- 디자이너 편집용 파일은 **관리자만 접근 가능**합니다.
- A+C 조합(PSD + SVG 모두 제공)도 가능합니다 (구현 5~6일).
