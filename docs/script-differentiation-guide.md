## 1. 카테고리별 규칙

### 1.1 식품/농축수산물

**관련 법규**: 식품표시광고법 제8조 제1항, 시행령 제2조·제3조, 농산물원산지표시법, 전자상거래 상품정보제공 고시 [출처 8]

**반드시 포함 (Required)**
- 식품의 명칭, 식품유형, 영업소 명칭
- 원재료명 및 함량 (함량 많은 순서대로)
- 원산지 표시 (농산물원산지표시법, 수산물은 해역 포함)
- 영양성분 표시 (열량, 탄수화물, 당류, 단백질, 지방, 포화지방, 트랜스지방, 콜레스테롤, 나트륨 등 9개 항목)
- 알레르기 유발물질 22종 표시 (함유량 무관, 반드시 표시)
- 소비기한 또는 품질유지기한, 제조연월일, 산란일
- 보관 방법 및 취급 주의사항
- 내용량 및 해당 열량(kcal)
- 제조업소명 및 소재지
- GMO 해당 시 표시, 식품이력추적관리 사항
- HACCP / 유기농 / GAP 등 인증 마크
- 반품/교환 안내
- **생산자 실명·사진** (업계 관행 — "얼굴 있는 먹거리" 신뢰 확보)
- **배송 포장 상태 촬영** (아이스박스, 보냉백 등 신선도 유지 증빙)

**절대 금지 (Prohibited)** — 식품표시광고법 제8조 [출처 8]

| 금지 표현 | 허용 대체 표현 |
|-----------|---------------|
| "당뇨 치료에 탁월한 효능" | "비타민C가 풍부한 과일입니다" |
| "혈관 속 피를 맑게 해주는 작용" | "아삭한 식감과 높은 당도(Brix)를 자랑합니다" |
| "A사 제품과 달리 화학첨가물 0%" | "본 제품은 신선한 원료로 제조되었습니다" |
| "암 예방", "혈압 낮춤" | 원물의 영양적 특성 객관적 서술 |
| "100% 유기농" (인증 없이) | 인증 획득 시 인증마크와 함께 표기 |
| "최고", "가장" (근거 없이) | 구체적 수치(당도, 중량 등)로 대체 |
| 타 업체 비방, 원산지 허위 | 자사 제품 강점만 팩트 기반 서술 |

**핵심 원칙**: 질병명(당뇨, 고혈압, 암 등)을 직접 언급하거나 신체 조직의 기능 개선을 단정적으로 주장하면 의약품/건강기능식품으로 오인 → 행정처분 1순위. 시행령 제3조에 따라 타사 제품과의 부당 비교·간접 비방도 금지. [출처 8]

**권장 섹션 구조** (전환 심리 기반) [출처 4, 7]
1. **시즐(Sizzle) 후킹** — 극강 클로즈업 컷 (육즙, 이슬 맺힌 원물, 조리 완성 플레이팅)
2. **산지 스토리** — 생산자 스토리, 농장/어장 청정 풍경 (소셜 프루프)
3. **핵심 USP 3가지** — 당도 측정 결과 사진, 중량 확인 컷 등 객관적 지표 (쿠팡 가이드라인: 3가지 이내 요약)
4. **활용 제안** — 레시피, 조리 예시로 사용 가치 부여
5. **법적 정보** — 영양성분 표시 테이블, 알레르기 정보
6. **인증 마크** — HACCP, 유기농, GAP 등
7. **배송/언박싱** — 냉장/냉동 배송 상태 촬영, 포장 디테일
8. **고객 리뷰 + 교환/반품**

**톤앤매너**: 농어부의 정직한 땀방울 + 자연의 선물 → "따뜻한 전문성". 감성적 스토리텔링 기반이되 당도·무게·인증마크 등 객관적 수치를 교차 사용.

**촬영 가이드**
- 매크로(Macro) 컷: 조리 전 원재료 싱싱함 (신선도, 색감, 질감)
- 시즐(Sizzle) 컷: 조리 후 완성 요리의 먹음직스러운 대비
- 색온도: 붉은빛/노란빛 따뜻한(Warm) 톤으로 식욕 자극
- 패키지 전면/후면 (표시사항 노출)
- 사이즈 비교컷 (손, 그릇 대비)
- 단면컷 (과일 단면, 고기 마블링 등)
- 산지/생산자 사진, 배송 포장 상태

### 1.2 건강기능식품

**관련 법규**: 건강기능식품에 관한 법률 제10조 제1항, 건강기능식품 표시기준, 식품표시광고법, 건강기능식품 기능성 표시·광고 심의 기준 [출처 8, 9, 11]

**반드시 포함 (Required)**
- "건강기능식품" 인증 마크 (식약처 도안) — 최상단 배치
- GMP 인증 마크 — 최상단 배치
- 기능성 내용 (식약처가 인정/고시한 문구만 사용)
- 일일 섭취량 및 섭취 방법
- 섭취 시 주의사항 (의약품 병용, 알레르기, 임산부/어린이 주의)
- **"본 제품은 질병의 예방 및 치료를 위한 의약품이 아닙니다"** 면책 문구 (법적 필수)
- 원료명 및 함량, 영양/기능 정보 표시
- 유통기한
- 광고 시 사전심의 필증번호
- **캡슐/정제 실제 크기 비교 컷** (동전 등 대비, 업계 관행)
- **식약처 기능성 인정 원료 확인서 사본** (이미지 첨부, 업계 관행)

**절대 금지 (Prohibited)** — 건강기능식품 심의 기준 [출처 11]

| 금지 표현 | 허용 대체 표현 |
|-----------|---------------|
| "치매 예방에 탁월" | "기억력 개선에 도움을 줄 수 있음" |
| "간염 개선", "당뇨 치료" | "간 건강에 도움을 줄 수 있습니다" |
| "의학박사 OOO 추천" | "의사 OOO가 R&D 과정에 공동 참여함" |
| "혈압을 낮춥니다" | "혈압이 높은 사람의 혈압 조절에 도움을 줄 수 있음" |
| "살이 빠집니다" / "100% 효과" | "체지방 감소에 도움을 줄 수 있습니다" |
| "소화장애는 명현반응입니다" | (사용 불가 — 부작용을 호전반응으로 위장 금지) |
| "병원/약국에서만 판매하는 전용 제품" | (사용 불가 — 유통채널로 품질 우수성 간접 주장 금지) |
| 부원료 기능성 과장 / 부원료 특허 노출 | 주원료 기능성만 식약처 인정 범위 내 서술 |

**3대 빈번 적발 유형** [출처 11]:
1. **부원료 과장**: "보조원료로 면역력 증가를 돕는 키토산 첨가" → 제품 전체 효능으로 오인 유발
2. **전문가 권위 차용**: 의사/약사/교수의 추천·보증·공인. 단, R&D 직접 참여 사실의 객관적 명시만 허용
3. **호전반응(명현반응) 주장**: 부작용을 체질 개선 과정으로 속여 지속 섭취 권장

**권장 섹션 구조** (전환 심리 기반) [출처 11]
1. **권위 선언** — 건강기능식품 인증마크 + GMP 마크 배치
2. **페인포인트 공감** — "자주 깜빡깜빡하시나요?" (질병명 대신 일상적 불편함으로 우회)
3. **핵심 기능성** — 식약처 인정 주원료 기능성을 공식 허용 문구로 서술
4. **과학적 증거** — 인체적용시험 결과 데이터를 시각적 그래프로 배치
5. **제형 디테일** — 흡수율 기술력, PTP 개별 포장 등 위생/안전 강조
6. **섭취방법 + 주의사항** (면책 문구 포함)
7. **인증서/시험성적서 이미지**
8. **고객 리뷰 + FAQ + 배송/교환/반품**

**톤앤매너**: 극도로 이성적·논리적·차분한 "전문가 톤". 인체적용시험 통계, 특허 번호, 정부 인증 마크를 전면에 배치. 감정 호소 최소화.

**촬영 가이드**
- 블루 톤/화이트 톤 깔끔한 배경 (신뢰감·위생)
- 실험실 콘셉트 연출 컷
- 캡슐/정제 접사 + 자(Ruler) 또는 손바닥 위 크기 비교
- 1일 섭취량 시각화
- 패키지 전면 (인증마크 노출) / 후면 (성분표, 주의사항)
- 원료 원물 이미지 (홍삼, 프로폴리스 등)
- GMP 제조시설 사진, 인증서/시험성적서

### 1.3 뷰티/화장품

**관련 법규**: 화장품법, 화장품 표시·광고 관리 가이드라인, 화장품 광고자문 기준 및 해설서(2025 개정) [출처 12, 13]

**반드시 포함 (Required)**
- 화장품의 명칭
- 전성분 표시 (함량 많은 순서, 표준화 일반명)
- 화장품 책임판매업자 상호 및 주소
- 사용기한 또는 개봉 후 사용기한
- 내용량, 가격, 제조번호
- 사용 시 주의사항
- 기능성 화장품: "기능성 화장품" 문구 + 심사/보고 받은 효능효과
- 화장품 유형
- **피부 자극 테스트 완료 성적서** (업계 관행)
- **Before & After 이미지** (동일 조건 촬영 문구 필수 명시)

**절대 금지 (Prohibited)** — 화장품 광고자문 기준 [출처 12, 13]

| 금지 표현 | 허용 대체 표현 |
|-----------|---------------|
| "여드름 박멸", "아토피 완치" | "민감한 피부를 위한 저자극 설계" |
| "피부 세포 재생" | "피부 컨디셔닝에 도움" |
| "주름 완전 제거" | "주름 개선에 도움을 줄 수 있음" (기능성 인증 시) |
| "영구적 미백" | "피부 톤 케어" / "미백에 도움" (기능성 인증 시) |
| "독소 배출" | (사용 불가 — 생리학적 변화 주장) |
| "절대 부작용 없는 100% 무독성" | "저자극 테스트 완료" |
| "천연", "오가닉" (인증 없이) | 식약처 천연 화장품 기준 충족 또는 외부기관 인증서 증빙 시 사용 가능 |
| "피부과전용", "약국전용" | (사용 불가) |

**핵심 원칙**: 피부 질환명(여드름, 아토피)을 사용한 치료·박멸 표현, 진피층 세포 "재생"이나 "독소 배출" 등 생리학적 변화 주장은 명백한 위법. 비건/천연 표기는 공인 인증서가 상세페이지 내에 시각적으로 증빙되어야 함. [출처 12, 13]

**권장 섹션 구조** (감성→이성 전환) [출처 7]
1. **시각적 후킹** — 고감도 모델 연출 컷 + 패키지 디자인 (브랜드 아이덴티티)
2. **임상 데이터** — "사용 4주 후 눈가 주름 OO% 개선", 피부 자극 테스트 완료 마크
3. **텍스처 체험** — 제형 점도, 끈적임 테스트, 발림성/흡수력을 GIF 동적 이미지로
4. **성분 안전성** — 특허 추출 방식, EWG 그린 등급, N-Free 리스트
5. **사용법** — 브랜드 독자 스킨케어 루틴, How to use 스텝
6. **전성분 목록 + 주의사항**
7. **고객 리뷰 + 배송/교환/반품**

**톤앤매너**:
- 더마 코스메틱: 흰색 가운, 임상 그래프 → 신뢰감 있는 "연구원 톤"
- 색조 화장품: 트렌드 선도, 감각적 → "스타일리시 톤"
- 공통: 세련되고 감각적 + 전문적, 성분 과학과 감성 브랜딩의 균형

**촬영 가이드**
- 초접사 질감 컷 (쫀쫀함, 텍스처)
- 피부 도포 시 광채(Glow) 영상 컷
- 핵심 성분 원물 연출 컷 (제주 녹차, 병풀 등)
- 제품 단독 컷 (깔끔한 배경)
- 스와치 컷 (피부 위 발색/도포)
- 모델 사용 컷 + 라이프스타일 컷
- 패키지 후면 (전성분 노출)

### 1.4 패션/의류/잡화

**관련 법규**: 전자상거래 상품정보제공 고시, 섬유제품 품질표시기준, 대외무역법, 상표법, 부정경쟁방지법 [출처 16]

**반드시 포함 (Required)**
- 섬유 조성/혼용률 (백분율, 오차 2% 이내)
- 치수(사이즈) 표시 (cm 단위)

**식품/농축수산물 × 카카오쇼핑**: 산지 스펙보다 **"선물로서의 가치"가 압도적으로 중요**. 고급스러운 패키지, 아이스박스 보냉 상태, 메시지 카드 동봉 여부를 최상단 뷰에 배치하여 감성적 호소 극대화.

**생활용품/가구 × 오늘의집**: 하얀 배경 누끼컷이나 스펙 나열은 전환율 하락 주원인. **실제 유저의 스타일링 샷(룸셋) + 비포/애프터** 이미지를 매거진 형식으로 배치.

**뷰티/화장품 × 카카오**: 선물하기 특성상 패키징 비주얼이 매우 중요. 기능성보다 감성적 브랜딩 우선.
