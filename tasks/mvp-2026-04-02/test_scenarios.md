## 테스트 범위

### 1. 법적 컴플라이언스 검증

이 문서는 `script-differentiation-guide.md` 기준으로 검증합니다.

#### 1.1 카테고리별 금지 표현 검사
- **식품/농축수산물 카테고리**
  - ❌ 질병명 언급 (당뇨, 고혈압, 암 등)
  - ❌ 신체 기능 개선 단정 ("혈관 속 피를 맑게 해주는")
  - ❌ 타사 제품 비방 ("A사 제품과 달리")
  - ❌ 근거 없는 최상급 표현 ("최고", "가장")
  - ❌ 원산지 허위 표시
  - ✅ 객관적 수치 제시 (당도, 중량, 영양성분)

- **건강기능식품 카테고리**
  - ❌ 질병 치료/예방 단정
  - ❌ 의약품과 혼동 유발 표현
  - ✅ 식약처 인정 기능성만 사용

#### 1.2 필수 포함 요소
- **식품**
  - ✅ 영양성분 표시 (열량, 탄수화물, 단백질 등 9개)
  - ✅ 알레르기 유발물질 22종 표시
  - ✅ 원산지 표시
  - ✅ 보관 방법 및 주의사항
  - ✅ 인증 마크 (HACCP, 유기농, GAP 등)

- **건강기능식품**
  - ✅ 건강기능식품 인증 마크 (식약처 도안)
  - ✅ GMP 인증 마크
  - ✅ "본 제품은 질병의 예방 및 치료를 위한 의약품이 아닙니다" 면책 문구
  - ✅ 기능성 내용 (식약처 인정 문구만)
  - ✅ 일일 섭취량 및 섭취 방법

### 2. 콘텐츠 품질 검증

#### 2.1 스토리텔링 및 구조
- [ ] 시즐(Sizzle) 후킹 - 극강 클로즈업 컷
- [ ] 산지 스토리 - 생산자 정보
- [ ] 핵심 USP 3가지 제시
- [ ] 활용 제안 - 레시피/조리 예시
- [ ] 법적 정보 섹션
- [ ] 인증 마크 표시
- [ ] 배송/언박싱 정보
- [ ] 고객 리뷰 + 교환/반품 정보

#### 2.2 톤앤매너
- [ ] "따뜻한 전문성" 느낌
- [ ] 감성적 스토리텔링
- [ ] 객관적 수치 교차 사용

### 3. 이미지/메타데이터 검증

#### 3.1 이미지 가이드라인 준수
- [ ] 매크로(Macro) 컷: 싱싱한 원재료
- [ ] 시즐(Sizzle) 컷: 완성 음식
- [ ] 따뜻한 톤(Warm)의 색온도
- [ ] 패키지 전면/후면 노출
- [ ] 사이즈 비교 컷 (손, 그릇)
- [ ] 단면 컷 (과일 단면 등)
- [ ] 산지/생산자 사진
- [ ] 배송 포장 상태 촬영

#### 3.2 플랫폼별 메타데이터
- [ ] 쿠팡: 상세페이지 3가지 USP 이내 요약
- [ ] 스마트스토어: 카테고리별 필수 표시 준수

## 테스트 케이스

### TC-001: 식품 카테고리 - 금지 표현 검사
**전제조건**: script.json 수령  
**테스트 단계**:
1. script 전문 읽기
2. 금지 표현 목록과 대조
3. 발견 시: violations 기록

**예상 결과**:
- 금지 표현 0개 → PASS
- 금지 표현 1개 이상 → FAIL

---

### TC-002: 건강기능식품 - 필수 문구 검사
**전제조건**: script.json 수령 (카테고리=건강기능식품)  
**테스트 단계**:
1. "본 제품은 질병의 예방 및 치료를 위한 의약품이 아닙니다" 존재 확인
2. GMP, 건강기능식품 인증마크 확인
3. 식약처 기능성 인정 문구만 사용되었는지 확인

**예상 결과**:
- 모든 필수 요소 포함 → PASS
- 1개 이상 누락 → FAIL

---

### TC-003: 인증 마크 검증
**전제조건**: script.json 수령  
**테스트 단계**:
1. 제품별 보유 인증 확인
2. script에 인증 마크 명시 여부 확인

**예상 결과**:
- 보유 인증 모두 명시 → PASS
- 누락된 인증 있음 → FAIL

---

### TC-004: 원산지 표시 검증
**전제조건**: script.json 수령 (카테고리=식품/농축수산물)  
**테스트 단계**:
1. 원산지 정보 포함 확인
2. 수산물인 경우 해역 포함 확인
3. 원산지 허위 표시 없음 확인

**예상 결과**:
- 원산지 명확히 표시 + 허위 없음 → PASS
- 원산지 누락 또는 허위 → FAIL

---

### TC-005: 영양정보 표시 (식품)
**전제조건**: script.json 수령 (카테고리=식품)  
**테스트 단계**:
1. 필수 9개 항목 포함 확인: 열량, 탄수화물, 당류, 단백질, 지방, 포화지방, 트랜스지방, 콜레스테롤, 나트륨
2. 알레르기 유발물질 22종 표시 확인
3. 보관 방법 및 주의사항 확인

**예상 결과**:
- 모든 항목 포함 → PASS
- 1개 이상 누락 → FAIL

---

### TC-006: 스토리텔링 구조 검증
**전제조건**: script.json 수령  
**테스트 단계**:
1. 권장 섹션 8가지 모두 포함 확인:
   - 시즐 후킹
   - 산지 스토리
   - 핵심 USP 3가지
   - 활용 제안
   - 법적 정보
   - 인증 마크
   - 배송/언박싱
   - 고객 리뷰 + 교환/반품

**예상 결과**:
- 8가지 모두 포함 → PASS
- 3개 이상 누락 → FAIL
- 1-2개 누락 → WARNING (권고사항 제시)

---

### TC-007: 타사 비방 표현 검사
**전제조건**: script.json 수령  
**테스트 단계**:
1. "A사", "경쟁사" 등 명시적 비방 확인
2. 간접 비방 (암시적 폄하) 확인

**예상 결과**:
- 타사 비방 0건 → PASS
- 1건 이상 → FAIL
